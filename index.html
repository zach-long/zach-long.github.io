Code

Code should be elegant.
What is elegant code?

I think elegant code makes things easier for both the user and the developer.


The example below is from a chat bot I built for a friend.

How does this code establish elegance?

  • This module draws data from a private config.json established by the user. This allows the user to set their own parameters for the server like name, IP, and port, without needing to put it in the code.
  • This particular module is where bot commands are established. The commands Object acts as a controller for this. Using the "status" Object as an example, we can see:
    • "name": The property of this is what users - people who can communicate with the bot - type in order to execute the command.
    • "action": This is the name of a function associated with the action.
    • "description": This is the description given to users when they might ask what the bot can do.

This design makes it easy for a developer to link the experience of the user, to the logic of the program.
More importantly, it also sets a precedent for how additional features can be added to the bot, and makes it simple to do so.


            const conf = require('./config.json');
            const execStatus = require('./helperFunctions').status;
  
            module.exports.commands = {
              "docsLink": {
                "name": "docsLinkPls",
                "action": docsLink,
                "description": "Sends the link to the discord.js api documentation."
              },
              "ip": {
                "name": "ip",
                "action": ip,
                "description": "Broadcasts the ip address of the server."
              },
              "status": {
                "name": "status",
                "action": status,
                "description": "Says whether or not the server is up, and how many people are online."
              }
            }
  
            function docsLink(args, cb) {
              let str = `View API documentation here: https://discord.js.org/#/docs/main/stable/general/welcome`;
              cb(null, str);
            }
  
            function ip(args, cb) {
              let str = `${conf.discord_server_name}: ${conf.ip}:${conf.port}`;
              cb(null, str);
            }
  
            function status(args, cb) {
              execStatus().then((result) => {
                console.log(`* commands.js > status() > result = ${result}`);
                cb(null, result);
              }).catch((err) => {
                console.log(`* commands.js > status() > err = ${err}`);
                cb(err);
              });
            }
